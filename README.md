# AI Desktop Assistant

## Overview
A multimodal AI assistant capable of:
- Voice interaction
- Object detection
- OCR text reading
- System automation (apps, files, browser)

## Features
- Wake word detection ("Jim")
- Speech-to-text using Whisper
- Object detection using YOLOv8
- OCR using Tesseract
- File and app control via voice
- Browser automation with Selenium
- Desktop automation with PyAutoGUI

## Example Commands
- "Open YouTube and play MrBeast"
- "Open file named resume"
- "Search Google for AI trends"
- "Open Notepad and write hello world"

## Tech Stack
- Python
- Whisper
- YOLOv8
- Tesseract OCR
- Selenium
- PyAutoGUI
